# MDPV is Markdown preview service

## Demo site

http://mdpv.herokuapp.com/

## How to deploy to Heroku

Get Heroku account.

Install Heroku Toolbelt.

```
$ heroku login
$ git clone https://github.com/junsuke39/mdpv.git
$ cd mdpv
$ heroku create
$ git push heroku
$ heroku open
```

## Supported browser

Google Chrome
